# Fonts Directory

This directory contains custom fonts for the E-Commerce Pro Platform.

## Font Requirements

### Primary Font: Inter
- **Source:** Google Fonts (https://fonts.google.com/specimen/Inter)
- **Weights:** 300, 400, 500, 600, 700
- **Usage:** Main body text, headings, UI elements

### Secondary Font: (Optional)
Consider adding a secondary font for:
- Logo text
- Special headings
- Marketing text

## Setup Instructions

### Method 1: Using Google Fonts CDN (Recommended)
Add to your HTML head:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap" rel="stylesheet">

# E-Commerce Frontend

This is the frontend React application for the E-Commerce Platform.

## Features
- Modern React with Hooks
- Context API for state management
- Responsive design
- SEO optimized
- Payment integration (Stripe, PayPal, Airtel Money)

## Installation
1. `npm install`
2. `npm start`

## Environment Variables
Copy `.env.example` to `.env` and update values.
